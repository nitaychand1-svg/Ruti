# QuantAI Symbiosis v1.0

Симбиоз квантовых фондов (RenTec, Two Sigma, Citadel и др.) на базе ACTS v6.0.

## Установка
pip install -r requirements.txt

## Запуск
python acts_v6.py

## Описание
- Phase 1: Weak signals (RenTec).
- Phase 2: Multi-manager agents + stat arb (Citadel, D.E. Shaw).
- Phase 3: Low-latency execution (HRT).
- Phase 4: Deep ML optimization (Man Group).

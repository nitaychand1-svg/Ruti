# Перенос истории чата из Claude в Cursor

Этот скрипт помогает перенести историю разговоров из приложения Claude в Cursor.

## Способы получения истории из Claude

### Способ 1: Экспорт из веб-версии Claude
1. Откройте https://claude.ai в браузере
2. Откройте нужный разговор
3. Скопируйте весь текст разговора (Ctrl+A, Ctrl+C)
4. Вставьте в текстовый файл и сохраните как `claude_chat.txt`

### Способ 2: Использование API (если доступно)
Если у вас есть доступ к API Claude, можно экспортировать через API.

### Способ 3: Ручное копирование
Просто скопируйте сообщения из приложения Claude и вставьте в текстовый файл.

## Использование скрипта

### Базовое использование

```bash
python transfer_claude_history.py --input claude_chat.txt --output cursor_history.md
```

### Конвертация из JSON (если доступен экспорт)

```bash
python transfer_claude_history.py --input claude_export.json --format json --output cursor_history.md
```

### Автоматическое определение формата

```bash
python transfer_claude_history.py --input history.txt --output result.md
```

## Формат входных данных

### Текстовый формат
Скрипт поддерживает различные форматы текста:

```
User: Ваш вопрос здесь
Assistant: Ответ Claude здесь

Вы: Еще один вопрос
Claude: Еще один ответ
```

или

```
> Вопрос пользователя

Ответ Claude здесь
```

### JSON формат
Если доступен JSON экспорт, он должен содержать структуру вида:

```json
{
  "messages": [
    {
      "role": "user",
      "content": "Вопрос пользователя"
    },
    {
      "role": "assistant",
      "content": "Ответ Claude"
    }
  ]
}
```

## Использование результата в Cursor

После конвертации:

1. Откройте файл `cursor_history.md` в Cursor
2. Скопируйте нужные части истории
3. Вставьте в новый чат в Cursor для контекста

Или используйте файл как справочный материал для контекста в вашем проекте.
